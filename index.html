= vs ==

Sebastian

The difference between = and == is simple once the meaning of each is understood. The '==' operator will compare the value of the variable on the left with the value of the right. If they are of the same value, then the result of the use of that operator is true, otherwise it is false. The '=' operator assigns the value of the variable on the right to the variable on the left.

Parameter vs Argument

Sebastian

Parameters and arguments are often used interchangably, but there is a difference between them. Arguments are values passed into a function call, and parameters are variables defined in the function to receive them. A parameter represents a value that the procedure expects you to pass when you call it. The procedure's declaration defines its parameters. An argument represents the value that you pass to a procedure parameter when you call the procedure. The calling code supplies the arguments when it calls the procedure. Taken from Microsoft's documentation for Visual Basic but the definition stands true for Java also.
